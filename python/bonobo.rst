Bonobo (pre-alpha, work in progress)
====================================

Bonobo is an extract-transform-load toolkit for python 3.5+, aiming for simplicity.

Bonobo is a rewrite / cleanup of rdc.etl for Python 3.5+. It's a work in progress.
